    1. Название проекта: Автоматизированное тестирование учебного тренажера "Самокат" (https://qa-scooter.praktikum-services.ru/).

    2. Описание проекта: Набор автотестов для учебного тренажера "Самокат", проверяющий:
        1. Выпадающий список в разделе «Вопросы о важном». 
        2. Переход по логитипу Самокат
        3. Переход по логотипу Яндекс
        4. Заказ самоката с 2 наборами данных и входом с двух разных мест

    3. Как установить проект:

        Шаг 1. Клонирование репозитория

        git clone https://github.com/ananna84-cmd/Sprint_6.git
        cd SPRINT_6

        (переходит в папку проекта)

        Шаг 2. Установка зависимостей

        pip install -r requirements.txt

        (устанавливает все необходимые библиотеки)

    4. Как запустить проект:

    pytest tests/ --alluredir=allure_results 

    Для просмотр отчета :

    allure serve allure_results   

    <Ctrl+C> для выхода
